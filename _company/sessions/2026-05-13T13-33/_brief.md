# 📋 작업 브리프

**원 명령:** ### 1. 자체 개발 IDE

- 로컬 LLM 모델 연결 가능 + 필요시 외부 API를 활용한 LLM 모델 추가 연결
- VSCODE와 같은 코드 편집 및 폴더/프로젝트 관리
- AI를 통한 폴더/파일 생성 및 수정이 가능해야 함

### 2. AI 에이전트

- 여러 특화된 AI 에이전트를(.md) 생성해서 각 업무를 분업화 하여 상세하게 업무
- 기본 구조가 로컬 LLM 이므로 추가적인 비용 없이 컴퓨터만 켜져 있다면 24시간 돌릴 수 있음
- 목표를 정해두고 목표치에 도달 할 때까지 업무 지시가 없어도 자동으로 업무
- 그 외 매일매일 분석이나 할 일을 정해 놓고 정해진 시간 보고 필요

### 3. 외부에서 휴대폰으로 확인

- 텔레그램, 디스코드, 카톡 등 메신저 앱을 통해서 업무 지시가 가능해야 함
- 또한 지시해 놓은 것에 대해서 정해진 시간에 보고를 받음
- 그 외 필요한 것이 생기거나 중요한 것에 대한 결재가 필요한 경우 AI 에이전트가 알아서 판단해서 나에게 보고 및 승인
- 업무지시가 아니더라도 간단한 무언가를 물어보는 용도의 봇 기능 또한 필요
- 로컬 LLM 모델을 활용하다가 성능이 부족하다고 판단되면 AI 에이전트들이 나한테 보고 후 외부 API를 활용한 AI 모델 활용
- 개요
    
    ## 프로젝트 목표
    
    ```
    로컬 LLM 기반 AI 자동화 시스템 구축
    ```
    
    ## 핵심 기능
    
    ```
    - AI 에이전트 자동 업무 수행
    - 프로젝트 파일 자동 수정
    - 장기 실행 및 반복 작업
    - 메신저 보고 및 승인 시스템
    - 필요시 외부 API 활용
    - 추후 자체 IDE 개발


이런 프로젝트를 만들고 싶어 일단 첫걸음으로 AI LLM 모델들을 활용가능한 AI IDE부터 만들거야 그것을 위한 기획안 만들고 read.me 파일로 남겨줘
