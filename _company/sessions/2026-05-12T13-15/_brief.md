# 📋 작업 브리프

**원 명령:** 상태의 가시성

## 요약
사용자가 '상태의 가시성'이라는 키워드를 통해 UI/UX 디자인과 카피라이팅을 결합하여 시스템의 피드백 상태를 시각적으로 어떻게 보여줄지에 대한 구체적인 구현 전략을 요청했습니다.

## 분배
- **🎨 Designer**: 이전 작업에서 논의된 'Minimal Pulse' 컨셉을 바탕으로, 로딩/진행/에러/완료 상태를 각기 다른 애니메이션 피드백과 시각적 요직소로 차이화하여 구체적인 UI 디자인 가이드를 수립하라.
- **✍️ Writer**: 디자이너의 시각적 가이드를 바탕으로, 사용자에게 시스템 상태를 알리는 직관적이고 힙한 UX 카피라이팅(Minimal/Direct)을 최종 확정하라.
