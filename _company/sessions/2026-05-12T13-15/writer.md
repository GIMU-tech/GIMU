# ✍️ Writer: UX Copywriting Guide (Minimal Pulse)

## 🎯 Design Philosophy: "Less is More, Clarity is King"
우리는 사용자에게 '설명'하지 않고 '상태'를 보여줍니다. 문장은 짧고, 명확하며, 시스템의 흐름(Flow)을 방해하지 않아야 합니다.

---

## 1. [Loading] - The Thinking Pulse
*상황: AI가 깊은 생각을 하거나 데이터를 처리 중일 때*
**전략:** '기다려라'는 명령 대신, '지금 움직이고 있다'는 존재감을 전달합니다.

| Type | Copy (EN) | 카피 (KR) | Context |
| :--- | :--- | :--- | :---로딩/생각 중일 때 |
| **Default** | `Thinking...` | `생각 중...` | 가장 표준적인 상태 |
| **Active** | `Gathering bits` | `데이터를 모으는 중` | 데이터 로딩/수집 시 |
| **Deep** | `Brewing ideas` | `아이디어를 짜는 중` | 복잡한 로직 생성 시 |

---

## 2. [Processing] - The Active Pulse
*상황: 결과물이 조립되거나, 텍스트가 출력되는 과정*
**전략:** 작업이 진행 중임을 알리는 역동적인 피드백.

| Type | Copy (EN) | 카피 (KR) | Context |
| :--- | :--- | :--- | :--- |
| **Building** | `Assembling...` | `조립 중...` | 결과물 구조화 시 |
| **Writing** | `Typing...` | `쓰는 중...` | 텍스킹 생성 시 |
| **Syncing** | `Syncing...` | `맞추는 중...` | 데이터 동기화 시 |

---

## 3. [Error] - The Glitch Pulse
*상황: 오류가 발생하여 흐름이 끊겼을 때*
**전략:** 사용자를 탓하지 않되, 문제를 명확히 인지시키고 '해결 가능함'을 암시합니다.

| Type | Copy (EN) | 카피 (KR) | Context |
| :--- | :--- | :--- | :--- |
| **System** | `Lost connection` | `연결이 끊겼어요` | 네트워크/서버 이슈 |
| **Logic** | `Mind blank` | `생각이 멈췄어요` | AI 로직 오류 |
| **Action** | `Try again?` | `다시 시도할까요?` | 사용자에게 다음 액션 유도 |

---

## 💡 UX Writing Rules (The "Guani" Voice)

1. **No Politeness, Just Presence:** "~~입니다", "~~해주세요" 같은 불필요한 존댓말은 UI를 무겁게 만듭니다. 명사형이나 현재 진행형(~중)을 사용하여 담백하게 전달합니다.
2. **The Power of '...' (Ellipsis):** 우리는 말줄임표를 사용해 여백의 미를 유지합니다. 하지만 너무 남발하여 흐름을 끊어서는 안 됩니다.
3. **Directness:** 사용자가 무엇을 해야 할지 모를 때는 'Try again?'처럼 질문을 던져 다음 행동을 유도합니다.
4. **Tone Consistency:** 모든 카피는 '차갑지만 스마트한(Cool but Smart)' 톤을 유지합니다.

# 📋 UX Copy Summary (Final Draft)

## 🎯 핵심 전략
- **Minimalism:** 불필요한 수식어를 제거하고 '상태' 그 자체만 전달.
- **Directness:** 사용자가 즉각적으로 상황을 이해할 수 있는 직관적 단어 선택.

## 🛠️ 카피 라이브러리

### 1. Loading (생각 중)
- `Thinking...` (생각 중...)
- `Gathering...` (모으는 중...)

### 래선로 2. Processing (작업 중)
- `Building...` (만드는 중...)
- `Writing...` (쓰는 중...)

### 3. Error (오류 발생)
- `Oops, lost connection` (연결이 끊겼어요)
- `Something went wrong` (문제가 생겼어요)
- `Try again?` (다시 해볼까요?)
