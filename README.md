# 🚀 Project: Autonomous AI-Agent System (AAS)
> **"로컬의 자율성과 클라우드의 지능을 결합한 차세대 워크플로우"**

## 1. Project Vision
사용자의 개입을 최소화하면서도, AI 에이렉트들이 스스로 코드를 작성하고(IDE), 업무를 수행하며(Agent), 인간은 결정만 내리는(Mobile) **'자율 운영 비즈니스 엔진'** 구축을 목표로 합니다.

## 2. Strategic Goals (핵심 목표)
- **[Phase 1] AI-Native IDE:** AI가 파일 시스템을 직접 제어하고 맥락을 이해하는 지능형 작업 환경 구축.
- **[Phase 2] Autonomous Agents:** 24/7 중단 없는 업무 수행 및 목표 지향적 자율 실행 엔진 개발.
- **[Phase 3] Mobile Decision Loop:** 메신저를 통한 인간-AI 간의 유연한 인터랙션(지시/보고/결재) 구현.

## 3. Core Functionalities (핵심 기능 정의)

### 🛠️ Module A: AI-Native IDE (The Workspace)
*에이전트와 인간이 공유하는 지능형 작업장*
- **Hybrid LLM Engine:** 로컬 LLM(Llama 3, Mistral 등)을 기본으로 하되, 필요시 외부 API(GPT-4, Claude 등)로 즉시 전환 가능한 인터페이스.
- **Agentic File System:** AI 에이전트가 직접 파일을 생성, 수정, 읽고 쓰기 위한 정교한 권한 및 경로 관리 시스템.
- **Context-Aware Editor:** 코드 편집뿐만 아니라 프로젝트 전체의 구조(Tree)를 AI가 이해할 수 있는 데이터 구조로 제공.

### 🤖 Module B: Autonomous Agents (The Brain)
*목표를 향해 스스로 움직이는 팀*
- **Goal-Oriented Tasking:** 특정 목표가 주어지면 이를 하위 작업(Sub-tasks)으로 분해하고 실행하는 자율성.
- **Role-Based Specialization:** 각 에이전트가 `.md` 기반의 페르소나를 가지고 전문 영역(개발, 기획, 분석)에서 작업.
- **Automated Reporting:** 정해진 스케줄에 따른 작업 현황 보고 및 결과물 요약.

### 📱 Module C: Mobile Decision Loop (The Interface)
*언제 어디서나 가능한 통제권*
- **Messenger Integration:** 텔레그램/디스코드 등을 통한 실시간 업무 지시 및 보고 수신.
- **Decision Trigger:** 중요한 결정(결재)이 필요할 때만 인간에게 알림을 보내는 '효율적 개입' 시스템.
- **Human-in-the-loop:** AI가 성능 한계에 부딪히면 인간에게 판단을 요청하고, 인간의 승인 후 외부 API를 호출하는 지능적 루프.

## 4. Roadmap (로드맵)
1.  **Step 1 (Current):** AI-Native IDE 프로토타입 설계 및 파일 시스템 연동 구조 확립.
2.  **Step 2:** 로컬 LLM 기반 에이전트의 파일 생성/수정 기본 루프 구현.
3.  **Step 3:** 외부 메신저(Telegram 등) 연동 및 보고 체계 구축.
4.  **Step 4:** Full-Stack 자동화 및 모바일 결재 시스템 완성.
